📋 Como atualizar o site
  1. Clique em ⬇ Baixar JSON acima
  2. Substitua o arquivo data/content.json dentro da pasta do projeto
  3. Para cada imagem, veja o caminho exibido no campo (ex: assets/feed/IMG.png)
  4. Copie a imagem para a pasta indicada dentro de assets/
